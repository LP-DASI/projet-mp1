Installer et utiliser un framework CSS

Pour ce faire nous allons utiliser « la librairie HTML, CSS, JS la plus populaire du monde », Bootstrap.

Une fois sur la page des téléchargements 3 choix nous sont proposés :

  • Télécharger la version "compilée et minimisée" de Bootstrap qui est donc une version allégée du Framework.
  • Télécharger la version "Complète" du Framework comprenant les bases a l’utilisation de LESS CSS* ou la documentation de Bootstrap.
  • Porter Bootstrap vers SASS*.

Nous utiliserons la première option.

Dans l’archive nouvellement télécharger nous pouvons trouver l’arborescence suivante.

À l'intérieur du dossier CSS, nous trouvons le fichier bootstrap.css qui est la version CSS compilée, ainsi que le fichier bootstrap.min.css qui, en plus, est minimisé. Dans le dossier js sont placées les deux versions (compilée et minimisée) du fichier JavaScript de Bootstrap.

Afin de pouvoir utiliser Bootstrap dans votre site il vous suffit de déplacer le contenu de l’archive dans la racine de votre site web.
Le principe de bootstrap réside dans son layout en grille divisible en plusieurs parties. Chacune de ces parties est nommée container et il en existe 2 types :

  • Le .container, utilisé pour une longueur donnée.
  • Le .container-fluid, pour utiliser toute la longueur possible.

Chaque container peux disposer de une ou plusieurs lignes nommées row qui peuvent elles-mêmes disposer de une ou plusieurs colonnes nommées col.

Voici un exemple de code :

<div class="container">
 <div class="row">
  <div class="col-*"></div>
 </div>
 <div class="row">
  <div class="col-*"></div>
  <div class="col-*"></div>
  <div class="col-*"></div>
 </div>
 <div class="row">
 ...
 </div>
</div>

Les nombres à ajouter dans col-* doivent être entre 1 et 12.
