Qui est concerné par les Framework ?

Tous développeur web cherchant un design responsive paramétrable, un layout facilement manipulable ainsi qu’une structure prédéfinie pour le placement de chacun des éléments HTML de base.

Seulement il ne faut pas penser que les frameworks CSS sont tous équivalents et qu’ils sont dénués d’inconvénients…
