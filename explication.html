Qu'est-ce que c'est ?

Un Framework est un ensemble de composantes structurés conformément à un plan d’architecture.

Il crée et organise le code informatique pour faciliter le travail des programmeurs. Mettant en avant un « squelette » du programme.
Il simplifie la maintenance et améliore la productivité.

Il existe énormément de Framework qui ciblent différents langages web mais pas que : Java, HTML/CSS, Python, JavaScript, Perl…

  • Spring : Java
  • Bootstrap : HTML/CSS
  • ReactJS : Javascript
  • Django : Python
  • ...
