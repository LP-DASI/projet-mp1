Qu'est-ce que Pure CSS ?

Pure CSS est un framework responsive developpé par Yahoo. Ne pesant que 4.5KB compressé, Pure CSS est donc léger et rapide et peu par conséquent remplacer de plus gros framework dans de petits projets. Pure est comme la plupart des framework CSS léger construit sur les base de Normalize.css ce qui lui offre de nombreux avantages tels qu'une grille personnalisable et réactive ou encore un look propre et minimaliste qui peu facilement se personnaliser. De plus une grande quantitée de documentation est disponible sur le site de Pure CSS.

Avantage:

  • Léger
  • Simple d'utilisation
  • Grille personnalisable et responsive
  • Un look propre
  • Compatibilité
  • Coder uniquement en CSS
  • Bonne compatibilité avec les mobiles

Inconvénient

  • Pas de JQuery ou Javascript
