Qu’est ce que Bootstrap?

Bootstrap a été créé en 2011 chez Twitter par Mark Otto et Jacob Thorton comme solution interne au problème de structure de code au moment du développement de la plateforme. Chaque développeur codant à sa façon entraînant donc des incohérences dans les projets rassemblant beaucoup de développeurs.

Bootstrap est une compilation de plusieurs éléments et fonctions de web-design personnalisables. La dernière version de Bootstrap, fondé sur HTML5 et CSS3, est une bibliothèque totalement open-source sous licence MIT comprenant :

  • Du code tenant compte du format d’affichage des principaux outils de navigation(responsive design)
  • Des plugins jQuery de qualité
  • Une compatibilité maximale entre les différents navigateurs
  • Une bonne documentation alimentée par la communauté
  • et bien d’autres…

Bootstrap est un framework très populaire qui a une évolution très rapide avec l’arrivée fréquente de nouvelles versions. C’est à la fois un avantage ( il évolue très vite ) mais aussi un inconvénient (code rapidement obsolète).

La version 3 de Bootstrap a amené beaucoup de changements, comme la refonte totale du système de grille. Si vous souhaitez mettre à jour votre site de Bootrap 2 vers Bootstrap 3 il vous faudra obligatoirement réécrire ce site.
