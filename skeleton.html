Qu'est-ce que Skeleton ?

Skeleton est une librairie CSS permettant la mise en place d'un design responsive pour une application web. Par rapport à d'autre framework comme Bootstrap ou Foundation, Skeleton possède l'avantage d'être léger. Le secret de cette légèreté réside dans le fait que Skeleton à été créer uniquement en CSS et donc sans aucun Javascript, le tous contenu en 400 lignes. Pourtant ce qui se présente comme un avantage est aussi un défaut. Skeleton est donc destiné à être utilisé dans de petits projets web et deviendra rapidement limté dans des gros projets.

Avantage:

  • Léger
  • Simple d'utilisation
  • Très bon responsive
  • Coder uniquement en CSS
  • Grande compatibilité

Inconvénient

  • Long à développer pour avoir une personalisation plus complète
  • Uniquement pour de petits projets
  • Un manque de CSS intégré
  • Peu de template
  • Ne convient pas au prototyping
